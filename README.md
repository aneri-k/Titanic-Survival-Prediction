# Titanic-Survival-Prediction
This project uses the Titanic - Machine Learning from Disaster dataset from Kaggle. The dataset contains information about passengers aboard the Titanic and is used for binary classification tasks. The objective is to predict whether a passenger survived the disaster.

**Dataset Characteristics** \
Training samples: 891 passengers \
Test samples: 418 passengers \
Target variable: Survived \
Problem: Classification 

**Source** \
Dataset from Kaggle's competition: https://www.kaggle.com/competitions/titanic
